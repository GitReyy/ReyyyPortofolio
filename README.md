## 🚀 Cara Menggunakan

### 1. **Membuka Portofolio**
```bash
- Buka file index.html di browser
- Atau gunakan live server di VS Code
```

### 2. **Mengkustomisasi Data Pribadi**

Edit bagian berikut di `index.html`:
- Nama dan title Anda
- Email, telepon, lokasi
- Social media links (GitHub, LinkedIn, Twitter)
- Pengalaman kerja dan pendidikan
- Deskripsi tentang diri Anda

### 3. **Mengubah Warna**

Edit variable CSS di `styles.css`:
```css
:root {
    --primary-color: #667eea;    /* Ubah warna utama */
    --secondary-color: #764ba2;  /* Ubah warna secondary */
    --accent-color: #f093fb;     /* Ubah warna accent */
}
```

### 4. **Menambah Skill Baru**

Di section Skills, tambahkan skill category baru atau skill item:
```html
<div class="skill-item">
    <div class="skill-header">
        <span>Nama Skill</span>
        <span class="level">XX%</span>
    </div>
    <div class="progress-bar">
        <div class="progress" style="width: XX%"></div>
    </div>
</div>
```
